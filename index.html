模块化，接口化，插件化。

GameBox是Unity开发中间件。GameBox将各种模块定义成一个个服务，采用插件框架动态加载，通过抽象化的接口对外暴露。

version 1.0

第一步


                        将下载的ServiceCenter.Editor.dll放在Unity项目中的Editor目录

第二步


                        在Unity的GameBox菜单下选择ServiceCenter，在弹出的对话框中点击Create

第三步


                        在更新的服务列表中选择需要的模块，然后点击Update

第四步


                        在Hierarchy下创建一个GameObject，展开GameBox/ServiceCenter模块，将ServicePlayer挂到该GameObject上
                    

                        在Project下展开需要的模块文件，将对应的Installer挂在之前的GameObject上

变量

变量允许我们单独定义一系列通用的样式，然后在需要的时候去调用。所以在做全局样式调整的时候我们可能只需要修改几行代码就可以了。

                            // LESS
@color: #4D926F;
#header {
  color: @color;
}
h2 {
  color: @color;
}
 
/* 生成的 CSS */
#header {
  color: #4D926F;
}
h2 {
  color: #4D926F;
}
